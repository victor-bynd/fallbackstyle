## Development

This project uses Vite + React + TailwindCSS.

### Prerequisites

- Node.js 18+ 
- npm or yarn

### Installation

\`\`\`bash
# Install dependencies
npm install

# Start development server
npm run dev
\`\`\`

Visit `http://localhost:5173` to view the app.

### Production Build

\`\`\`bash
# Build for production
npm run build

# Preview production build locally
npm run preview
\`\`\`
